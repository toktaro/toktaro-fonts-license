# toktaro fonts license

toktaro fonts（とくたろフォント）が公開・販売しているフォントに関するライセンスを管理するリポジトリです。

## 対象フォント
本ライセンスは以下のフォントに適用されます。

- **アングル**
- **ニコイチ**
- その他、toktaro fontsから配布されるフォント
